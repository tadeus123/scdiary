# Office Gallery

Add images here and reference them in `config.json`.

## Config format

Each category has an `items` array. Each item:

```json
{
  "filename": "my-image.jpg",
  "title": "Product name",
  "description": "Why it's great"
}
```

- **filename**: Image in this folder (public/images/office/)
- **url**: Or use full URL for external images
- **title**, **description**: Optional
